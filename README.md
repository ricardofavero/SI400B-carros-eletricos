**Motivação (por que o projeto é relevante?):**  
> Porque a venda de carros elétricos nos EUA está em ascensão, o que cria uma demanda por postos de recarga.  
 
**Fonte(s) de dados (identificação da fonte e URL para os dados):**  
> [U.S. Department of Energy](https://catalog.data.gov/dataset/alternative-fuels-data-center) 
[Alternative Fuels Data Center](http://www.afdc.energy.gov/data_download) 
 
**Descrição dos dados (atributos ou campos presentes e o significado de cada um deles):** 
> Para indicar a localização dos postos de recarga mais próximos, o usuário poderá inserir um endereço com número, rua, cidade, estado ou CEP e selecionar quais opções de distancia deseja; ou escolher para que sua localização seja identificada pelo programa. Após escolhidos os dados de entrada, o programa irá apresentar os postos com: o nome da estação, endereço, cidade, estado, CEP, telefone, latitude e longitude, distância, horário de funcionamento e uma observação (se existir). 
 
**Volume de dados (quantos valores existem ou dimensão do conjunto de dados em bytes):** 
> 14148 postos de recarga (cada um com 10 tipos de dados diferentes). Arquivo .csv com 4.18MB.
